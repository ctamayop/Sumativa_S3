## 📘 Descripción general del sistema
LlanquihueTour es una aplicación desarrollada en Java que permite representar personas relacionadas con una agencia de turismo mediante el uso de Programación Orientada a Objetos (POO).

## Conceptos Aplicados
- Encapsulamiento mediante atributos privados.
- Herencia entre Persona, Cliente y Colaborador.
- Composición mediante las clases Rut y Direccion.
- Sobrescritura del método toString().
- Manejo de excepciones personalizadas.
- Uso de constructores, getters y setters.

```plaintext
